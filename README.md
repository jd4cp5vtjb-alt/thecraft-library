# thecraft-library
TheCraft Library - Votre bibliothèque de matériaux d’architecture Organisez vos matériaux et fournisseurs préférés. Parcourez par catégories (Bois, Pierre, Métal…), ajoutez vos coups de cœur et accédez aux infos essentielles : prix, descriptions, contacts. Interface moderne et intuitive pour architectes et designers. Importez/exportez vos données.
